### sklearn 学习

- 安装

```
window下下载easy_install到https://pypi.python.org/pypi/setuptools/下周压缩包
然后执行python setup.py install 
（可以通过virtualenv 来安装 一些包）
然后可以直接用 easy_install scikit-learn 安装 scikit-learn 了

```
